NQueens Genetic Algorithm Search
Created by Colin Hughes for CS481

Python-based UI that attempts to solve any size of NQueen chess puzzle using a genetic search algorithm.

--Instructions
-Run main.py
-Input desired parameters for genetic search in the text fields at the top (default values are given)
-Click RUN SEARCH button
-Wait for search to complete (progress is tracked in the command window)
-New search can be performed after results appear
